🎮 ContaminaDos — Social Deduction Game (v1.0.0)

ContaminaDos is an interactive social deduction game where players must collaborate, analyze behaviors, and make strategic decisions to uncover who the impostors are before it’s too late.

Throughout the game, players will face challenges, voting rounds, and critical situations where communication and logic are key to surviving and winning.

🚀 Features

🕵️‍♂️ Hidden roles: regular players and impostors with different objectives.

🗳️ Voting system to eliminate suspicious players.

🤝 Multiplayer interaction that encourages cooperation and deception.

🧩 Deduction and strategy mechanics based on observation and logic.

📱 Intuitive and responsive interface, adaptable to different devices.

☁️ Cloud deployment, ensuring availability and scalability.

🛠️ Technologies Used

Frontend: Angular, TypeScript

Backend: .NET

Cloud Infrastructure: Oracle Cloud Infrastructure (OCI)

Architecture: Modern and scalable web application
